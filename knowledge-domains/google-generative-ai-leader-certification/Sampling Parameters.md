# Sampling Parameters

Sampling Parameters 用于控制模型输出的随机性和多样性。

详细解释：

官方补充材料里明确提到 parameter playground，因此虽然 Leader certification 不考代码，仍建议理解 temperature、top-k、top-p 这类参数的作用。它们影响输出是否更保守、更稳定或更多样。

常见参数：

- temperature
- top-k
- top-p

考试层面要会：

- temperature 高，输出通常更发散
- temperature 低，输出通常更稳定
- 参数调优属于 prompt experimentation 的一部分
