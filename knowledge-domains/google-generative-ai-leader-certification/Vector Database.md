# Vector Database

Vector Database 是专门存储和检索 embeddings 的数据库。

详细解释：

在 Leader certification 里，你需要知道 vector database 在架构中的位置，而不是内部索引算法细节。它通常用于 semantic search 和 RAG retrieval。题目如果问“企业知识库问答最适合什么技术栈”，答案通常包含 vector database。

常见用途：

- semantic search
- RAG retrieval

例子：

- Pinecone
- Weaviate
- Vertex AI Vector Search
