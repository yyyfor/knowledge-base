# Google AI APIs

Google Cloud 的 GenAI offering 还包括一组可直接调用的 AI API。

详细解释：

study guide 中列出的 API 家族非常值得补充，因为它能帮助理解“Google AI offering 不只是 LLM”。这些 API 覆盖语音、视觉、文档和翻译场景，是企业把 AI 能力嵌入现有系统的常见入口。

常见 API：

- Speech-to-Text API
- Text-to-Speech API
- Translation API
- Cloud Vision API
- Document AI API
- Cloud Video Intelligence API
