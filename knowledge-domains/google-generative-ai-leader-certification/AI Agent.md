# AI Agent

AI Agent 可以概括为 LLM + tools + reasoning。

详细解释：

Agent 是 Module 5 的高频重点。考试不是要你写 agent 代码，而是理解 agent 为什么能够调用工具、访问外部系统、分步骤完成任务，以及它与单轮问答有什么区别。

Agent 能做：

- call APIs
- access databases
- execute tasks
