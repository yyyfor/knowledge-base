# LLM

LLM 是 Large Language Model，大语言模型。

详细解释：

它是最常见的 foundation model 之一，擅长文本理解、生成、总结、改写、问答和一定程度的推理。Google Leader certification 不要求数学细节，但要求你理解 LLM 的强项、局限，以及为什么需要 [[Prompt Engineering]]、[[RAG]] 和 [[Grounding]]。

常见能力：

- text generation
- summarization
- question answering
- reasoning support

常见局限：

- [[Hallucination]]
- knowledge cutoff
- bias
- context limitation

相关：

- [[Foundation Model]]
- [[Prompt Engineering]]
- [[RAG]]
