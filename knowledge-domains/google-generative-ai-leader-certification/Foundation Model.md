# Foundation Model

Foundation Model 是在大规模数据上预训练、可适配多任务的通用模型。

详细解释：

考试里这是基础定义题。需要知道 foundation model 不是为单一任务训练出来的 narrow model，而是可以通过 prompting、fine-tuning 或工具组合适配多种下游任务。LLM、Gemini 这类模型都属于 foundation models。

关键点：

- pre-trained on broad data
- supports multiple downstream tasks
- can be adapted by prompting or fine-tuning
