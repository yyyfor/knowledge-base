# Four Primary GenAI Use Cases

Google 官方课程中最核心的框架之一，是把 GenAI 使用方式分成四类。

四种主要用途：

- Content creation
- Knowledge assistants
- Automation
- Insights

详细解释：

这四类经常直接出现在题干里。考试喜欢给一个业务场景，让你判断更偏内容生成、知识问答、流程自动化还是数据洞察。做题时先判断目标产出是什么，再选最合适的架构和产品。

典型例子：

- Content creation：文案、图像、代码草稿
- Knowledge assistants：企业知识问答、客服问答
- Automation：文档处理、工作流自动执行
- Insights：从报告或数据中提炼结论
