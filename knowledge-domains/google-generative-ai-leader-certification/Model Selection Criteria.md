# Model Selection Criteria

企业选择 GenAI 模型时，需要综合考虑性能、成本、延迟和上下文能力。

详细解释：

这类题是高频情景题。不要只看“模型是否最强”，而要看 use case：实时客服通常更在意 latency，研究分析更在意准确性，大文档任务会更在意 context window。

常见考量：

- performance
- cost
- latency
- context window

例子：

- real-time support -> latency first
- research analysis -> accuracy first
