# API Integration

API Integration 指应用通过 API 与模型或工具进行连接。

详细解释：

在考试语境中，API integration 通常与 agent、tool use 和企业系统集成一起出现。常考点包括 API library、API key、authentication 和 authorization 的基本角色。

核心概念：

- API library
- API key
- authentication
- authorization
- model inference
