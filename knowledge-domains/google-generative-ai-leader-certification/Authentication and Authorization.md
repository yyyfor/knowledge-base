# Authentication and Authorization

Authentication 是确认“你是谁”，Authorization 是确认“你能做什么”。

详细解释：

在 API 和 enterprise agent 场景里，这两个概念经常一起出现。考试一般不会考协议细节，但会考它们在安全调用模型、数据源和工具时的作用区别。
