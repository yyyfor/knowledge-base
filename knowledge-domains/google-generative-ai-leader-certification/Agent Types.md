# Agent Types

官方 study guide 提到 agent 可以按实现方式区分为 deterministic、generative 或 hybrid。

详细解释：

这有助于理解并不是所有 agent 都完全依赖 LLM。某些 agent 更偏固定规则流程，某些更偏生成式推理，而 hybrid agent 则会把规则系统和 LLM 结合。

常见类型：

- deterministic agents
- generative agents
- hybrid agents
