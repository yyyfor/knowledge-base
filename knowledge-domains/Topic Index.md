## Quick Reference

### Algorithms by Time Complexity
**O(1) - Constant Time:**
- Hash table operations (average case)
- Array access by index

**O(log n) - Logarithmic Time:**
- [[Binary Search]]
- Heap operations

**O(n) - Linear Time:**
- Array/string traversal
- Tree/graph traversal (DFS/BFS)
- Hash table operations (worst case)

**O(n log n) - Linearithmic Time:**
- Efficient sorting algorithms
- [[Top K Problems]] with heap

**O(n²) - Quadratic Time:**
- Nested loops
- Brute force approaches
- [[Dynamic Programming]] with 2D DP

**O(2ⁿ) - Exponential Time:**
- [[Backtracking]]
- Recursive solutions without memoization
