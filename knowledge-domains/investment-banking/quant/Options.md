# Options

期权赋予买方在未来某一时间或期间，以约定价格买入或卖出标的资产的权利，而非义务。

详细解释：

Options 是最核心的非线性衍生品。买方支付 premium 获得上行收益或下行保护，卖方收取 premium 但承担或有风险。期权价值通常由 intrinsic value 和 time value 组成，因此它同时对标的价格、波动率、时间、利率等多个维度敏感。

常见分类：

- Call vs Put
- European vs American
- Vanilla vs Exotic
- Physical settlement vs Cash settlement

实务重点：

- 隐含波动率和 [[Volatility Surface]] 对期权定价极其重要
- 期权风险管理通常围绕 [[Greeks]] 展开
- 期权产品是 [[Black-Scholes Model]]、[[Binomial Tree Model]]、[[Monte Carlo Simulation]] 的典型应用场景
