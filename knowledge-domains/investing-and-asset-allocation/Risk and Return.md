# Risk and Return

收益从来不是免费的，风险是换取长期回报的重要来源。

详细解释：

投资时不能只看预期收益，更要看波动、回撤、流动性风险、信用风险和尾部风险。真正重要的是风险调整后的回报，以及这些风险是否适合你的目标和承受能力。

重点主题：

- expected return
- volatility
- drawdown
- risk premium
- risk-adjusted return
