# SQL and Databases for Quants

量化程序员几乎一定要会 SQL 和基本数据库设计。

详细解释：

交易、市场数据、因子、风险结果和回测结果通常都不只存在于 Python 内存里。不会 SQL 的量化程序员很难高效查询、验证和组织数据，更难搭建稳定研究和生产链路。

重点主题：

- joins and aggregations
- window functions
- schema design basics
- time-series querying
- data integrity and indexing
