# Linux and Developer Tooling

很多量化开发环境最终都运行在 Linux 和命令行工具链上。

详细解释：

即使你的研究起步于 notebook，生产系统、回测集群、CI、容器和远程服务器通常都离不开 Linux。熟悉 shell、进程、日志、权限、包管理和基础网络，会显著提高开发效率。

重点主题：

- shell basics
- process and file management
- logs and monitoring
- ssh and remote workflow
- environment management
