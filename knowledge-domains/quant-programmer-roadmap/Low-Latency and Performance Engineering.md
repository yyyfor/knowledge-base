# Low-Latency and Performance Engineering

如果工作接近交易、做市或高频系统，性能工程是核心竞争力。

详细解释：

这部分关注 CPU cache、内存布局、网络延迟、锁竞争、序列化开销和性能剖析。即使不做 HFT，很多 risk/pricing 系统也需要严肃的性能优化。

重点主题：

- memory layout
- concurrency
- profiling
- networking basics
- latency measurement
