# Machine Learning for Quants

机器学习对量化程序员有价值，但应该放在扎实基础之后。

详细解释：

很多岗位并不要求深度学习研究能力，但会要求理解监督学习、特征工程、交叉验证、模型评估和过拟合控制。对交易和研究工作来说，数据质量和实验设计常常比模型复杂度更重要。

重点主题：

- supervised learning basics
- feature engineering
- validation
- overfitting control
- model evaluation
