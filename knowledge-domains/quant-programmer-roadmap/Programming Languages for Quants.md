# Programming Languages for Quants

量化程序员通常至少要有一门高生产力语言和一门高性能语言。

详细解释：

最常见组合是 Python + C++。Python 用于研究、数据处理、原型和 glue code；C++ 常用于低延迟、数值密集或核心定价组件。SQL 也几乎是必需技能。

常见技术栈：

- Python
- C++
- SQL
- occasionally Java or Rust
