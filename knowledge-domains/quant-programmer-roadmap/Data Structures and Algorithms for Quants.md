# Data Structures and Algorithms for Quants

量化程序员仍然需要扎实的基础算法能力。

详细解释：

无论是面试、回测框架、订单簿处理还是分布式计算任务拆分，基础数据结构和算法能力都很重要。这里不是只为了刷题，而是为了写出复杂度可控、性能可预测的程序。

重点主题：

- arrays, hash maps, trees, heaps
- sorting and searching
- graph and dynamic programming basics
- complexity analysis
