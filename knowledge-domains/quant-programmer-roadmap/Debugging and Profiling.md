# Debugging and Profiling

量化程序员的价值不只在于写代码，也在于快速定位问题。

详细解释：

真实系统里常见的问题包括数据错位、数值不稳定、性能退化、交易版本不一致和环境差异。debugging 是把错误缩小到可解释范围，profiling 是找到真正的性能瓶颈，而不是靠猜。

重点主题：

- isolate the failure
- inspect inputs and assumptions
- profile CPU and memory
- verify numerical stability
- compare expected vs actual results
