# API Gateway and Service Boundaries

服务边界和 API gateway 设计决定系统如何演化。

详细解释：

设计服务时，核心不是拆得越细越好，而是边界是否清楚、接口是否稳定、责任是否单一。API gateway 常用于统一入口、认证、限流和路由。
