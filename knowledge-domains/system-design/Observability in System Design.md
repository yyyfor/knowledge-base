# Observability in System Design

可观测性决定你能不能真正运营一个系统。

详细解释：

很多设计题只讲功能和扩展，却忽略日志、指标、trace、告警和审计。没有可观测性，系统出问题时你只会知道“坏了”，却不知道“为什么坏”。
