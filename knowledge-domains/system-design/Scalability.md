# Scalability

Scalability 是系统在负载增长时仍能维持服务能力的能力。

详细解释：

扩展可以是纵向扩展，也可以是横向扩展。system design 面试里，scalability 通常意味着你要考虑流量增长、数据增长、热点问题和服务拆分，而不是只说“加机器”。
