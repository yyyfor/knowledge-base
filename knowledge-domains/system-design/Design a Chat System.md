# Design a Chat System

聊天系统是面试中很常见的“实时 + 状态 + 多端同步”场景。

核心关注：

- connection management
- message delivery
- ordering and retries
- online presence
- group chat scaling

为什么它有代表性：

它同时考察实时通信、存储、状态同步和高可用设计。
